The web, or World Wide Web (WWW), is a system of interconnected documents and resources accessed via the Internet. Created by Tim Berners-Lee at CERN in 1989, it functions by using HTTP, HTML, and URLs to link and display information through browsers. The web is a specific application that runs on top of the Internet, which is the underlying global network of computers.
